# 🧠 Build Your Own Personal AI Agent — The Handbook

### Build an AI "twin" that talks like you, lives on your chat app, and does real tasks — in 3 hours.

> **No coding needed.** You paste prompts → your AI tool (Claude Code or Codex) writes the code → you run a command to test. That's it.

> 🤖 **What you're building:** a **chat agent — a bot** that messages like you and lives on your chat apps (Telegram / Slack / Discord). That's the foundation. Extra powers — **Voice, reminders, web answers, memory, and more** — are *capabilities* you bolt on in Phase 4. **Voice is just one of them**, not the point.

---

## What you'll walk out with

- 🗣️ A **twin** that replies the way *you* talk
- 💬 Live on **Telegram, Slack, or Discord** — replies even when you're away
- 🧠 **Remembers** chats and facts you tell it
- ⚡ **Capabilities** you pick (reminders, web answers, voice, memory…)
- 💾 Saved with checkpoints — you never lose progress

---

## 📖 How to read this

**Pick your language first** — use the 🐍 **Python** / 🟨 **JavaScript** toggle at the top of the page. Every command and code block switches to match your choice.

1. Go **in order**, top to bottom.
2. Copy each **📋 PROMPT** into your AI tool; run each **💻 RUN** command.
3. Stuck or out of quota? Use the **🛟 fallback code** in that step.
4. Follow your **🟣 Claude** or **🟢 Codex** box where they differ.

> 🔋 Use **Sonnet** (`/model sonnet`) and one prompt per step so you don't hit your usage limit. See *How This Handbook Works*.
